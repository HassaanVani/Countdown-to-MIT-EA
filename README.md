# MIT Early Action · Class of 2030

A sophisticated, minimalist Vue.js application featuring a real-time countdown to MIT's Early Action deadline and a comprehensive application checklist. Designed with MIT's official brand colors and professional aesthetics.

## Features

- ⏰ **Live Countdown Timer**: Real-time countdown to November 1, 2026 (11:59 PM EST)
- ✅ **Interactive Checklist**: Track your progress with 14 essential application tasks
- 💾 **Progress Persistence**: Your checklist progress is saved in local storage
- 🎨 **Sophisticated Design**: Minimalist dark theme with MIT's official cardinal red (#A31F34)
- 📱 **Fully Responsive**: Seamless experience on desktop, tablet, and mobile
- ✨ **Sleek Animations**: Smooth transitions and hover effects
- 🎯 **MIT-Focused**: Tailored specifically for MIT Early Action applicants

## Tech Stack

- **Vue 3** - Progressive JavaScript framework
- **Vite** - Next-generation frontend build tool
- **TailwindCSS** - Utility-first CSS framework
- **LocalStorage API** - For saving checklist progress

## Application Checklist

The checklist includes all essential tasks:

1. Create MIT Applicant Account
2. Complete Common App/Coalition App Profile
3. Write Personal Essay
4. Complete MIT-Specific Essays
5. Request Teacher Recommendations
6. Request Counselor Recommendation
7. Take Standardized Tests
8. List Activities & Honors
9. Complete Self-Reported Coursework
10. Prepare Portfolio (if applicable)
11. Interview Preparation
12. Review & Proofread Everything
13. Submit Application Fee
14. Submit Application

## Important Dates

- **Application Deadline**: November 1, 2026 at 11:59 PM EST
- **Decision Release**: Mid-December 2026

## Tips for Success

- Start essays early (5-10 drafts recommended)
- Request recommendations 4-6 weeks before deadline
- Show passion, collaboration, and hands-on making
- Be authentic and specific in your application
